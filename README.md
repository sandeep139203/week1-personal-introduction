# Personal Introduction Program

## Project Description

This is a beginner Python project created as part of Week 1 learning.

The program asks users for their personal information and displays a friendly welcome message using the information entered.

## Features

- Accepts user input
- Stores information in variables
- Uses f-strings
- Displays formatted output
- Beginner-friendly code structure

## Technologies Used

- Python 3

## How to Run

1. Install Python.
2. Open terminal or command prompt.
3. Navigate to the project folder.
4. Run:

```bash
python personal_intro.py
```

## Example

Input:

Name: Alex
Age: 21
City: Hyderabad
Hobby: Coding

Output:

Welcome Alex!
You are 21 years old.
You live in Hyderabad.
You love Coding.
